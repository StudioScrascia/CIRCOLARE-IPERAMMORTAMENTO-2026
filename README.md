## Novità versione 2.0 (14/07/2026)

Aggiornamento completo con le disposizioni attuative del **DM 7.5.2026** e le modifiche del **DL 27.3.2026 n. 38 convertito**:

| Novità | Dettaglio |
|---|---|
| **DM 7.5.2026** | Disciplina compiutamente la procedura di accesso, la documentazione e le cause di decadenza |
| **Procedura GSE in 3 fasi** | Comunicazione preventiva → conferma con acconto 20% → completamento entro 15.11.2028 |
| **Decorrenza agevolazione** | Dal periodo d'imposta in cui si trasmette la comunicazione di completamento al GSE |
| **Perizia asseverata obbligatoria** | Per tutti i beni senza eccezioni — eliminata la soglia € 300.000 per l'autodichiarazione |
| **Certificazione contabile** | Obbligatoria, rilasciata da revisore legale iscritto sezione A ex art. 8 D.Lgs. 39/2010 |
| **Eliminazione requisito territoriale** | L'agevolazione spetta a prescindere dal luogo di produzione del bene (salvo impianti fotovoltaici) |
| **Software SaaS esclusi** | Non agevolabili i canoni relativi a software in cloud e modelli "as a service" |
| **Coordinamento CPB** | La maggiorazione non rileva nella determinazione del reddito oggetto di concordato preventivo biennale |
| **Cause di decadenza** | Codificate all'art. 10 DM 7.5.2026 |
